GET request:

With this API, you can obtain a user's all previously searched places information , photos URL , and user's submitted review/note on that place in json format. You just need to send a valid GET request to the following URL. 

"http://placefinderapi.herokuapp.com/api"
Include all the following GET parameters when you request the URL:
  1. key -this is your API Developer Key
For obtaining a particular place info , include following optional parameter:
  1. searchplace -Name of the Place you want


GET request Example:

Following code is in Python3:

import requests
url="http://placefinderapi.herokuapp.com/api"
params={
    "key":"Your_Developer_API_Key",
    "searchplace":"name_of_the_place"(optional)
}
r=requests.get(url,params=params)
print(r.content)

POST request:

Searching a new place info and storing it in your place history via our API is very easy. You simply have to send a valid POST request to the URL shown below. Please make sure you are sending the data as the UTF-8 charset! 

"http://placefinderapi.herokuapp.com/api"
Include all the following POST parameters when you request the URL:
  1. key- this is your API Developer Key
  2. searchplace- Name of the place whose info and pics you want
  3. review- Review/Note you want to add about this place(OPTIONAL)

Few points to remember while sending a request:

  • Your place name should be new i.e. if you have already searched before about place then it is stored in your history and you can not make the same request again . Rather you can find the contents of the place using a GET request
  • Place name should be valid . If you write any invalid place name the information about the place will not be dislayed but you will be shown 5 pictures of places that resembles your place name .
  • Photos of places are shown by PlaceAutocomplte method . The Place Autocomplete service can match on full words as well as substrings. Applications can therefore send queries as the user types, to provide on-the-fly place predictions.
  • Writing a review is not compulsory .


POST request Example:

Following code is in Python3: 

import requests
url="http://placefinderapi.herokuapp.com/api"
data={
    "key":"Your_Developer_API_Key",
    "searchplace":"Name_of_Place",
    "review":"write_your_note"(OPTIONAL)
}
r=requests.post(url,data=data)
print(r.content)


PATCH request:

With this API you can update your Review/Note. You will need to send a valid PATCH request to the URL below to update the data: 

"http://placefinderapi.herokuapp.com/api"
Include all the following PATCH parameters when you request the URL:
  1. key- this is your API Developer Key
  2. searchplace- Name of the place whose review you want to update
  3. review- Review/Note you want to set as new review


PATCH request Example:

Following code is in Python3: 

import requests
url="http://placefinderapi.herokuapp.com/api"
data={
    "key":"Your_Developer_API_Key",
    "searchplace":"Name_of_Place_whose_Review_is_to_updated",
    "review":"New_Review/Note"
}
r=requests.patch(url,data=data)
print(r.content)


DELETE request:

With this API you can delete your Place History , i.e. , all the data along with pictures and your review/note . You will need to send a valid DELETE request to the URL below to delete the data: 

"http://placefinderapi.herokuapp.com/api"
Include all the following DELETE parameters when you request the URL:
  1. key- this is your API Developer Key
  2. searchplace- Name of the Place


DELETE request Example:

Following code is in Python3: 

import requests
url="http://placefinderapi.herokuapp.com/api"
data={
    "key":"Your_Developer_API_Key",
    "searchplace":"Name_of_place",
    ""
}
r=requests.delete(url,data=data)
print(r.content)
